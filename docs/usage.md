# Usage Guide

This guide explains how to use the Financial Report Analyzer.

## Starting the Application

1. Open `index.html` in your browser.
2. Wait for the dashboard to load.

## Selecting a Company

- Use the dropdown menu.
- Choose a company like AAPL, MSFT, GOOGL, or AMZN.

## Viewing Financial Data

You can view:

- Revenue
- Net Income
- Earnings Per Share (EPS)
- Sentiment Score

Charts update automatically when you change company.

## Dark Mode

- Click the Dark Mode button.
- Switch between light and dark themes.

## Tabs

- Trends → Shows revenue growth
- Comparison → Compares companies
- Risks → Displays risk factors
- Sentiment → Shows sentiment score

## Export Charts

- Right-click on any chart.
- Select “Save Image As”.

## Tips

- Use desktop for best experience.
- Refresh page if data does not load.

## Limitations

- Only limited companies supported.
- No live data yet.
- No mobile app version.
