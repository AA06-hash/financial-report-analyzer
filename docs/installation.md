# Installation Guide

This guide explains how to install and run the Financial Report Analyzer.

## Requirements
- Any modern web browser (Chrome, Firefox, Edge)
- No internet connection required (for static version)

## Installation Steps

### Step 1: Download Project
Download or clone the project folder to your computer.

```bash
git clone https://github.com/AA06-hash/financial-report-analyzer.git

### Step 2: Open Folder

Go to the project folder:

cd financial-report-analyzer

### Step 3: Run Project

Open the file index.html in any web browser.

The dashboard will load automatically.
Folder Details

    index.html → Main dashboard

    styles.css → Design and layout

    scripts.js → Logic and charts

    data/ → Company JSON files

    docs/ → Documentation files

    images/ → Screenshots and diagrams
