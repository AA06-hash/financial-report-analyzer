# API Reference

This project uses static JSON files instead of live APIs.

## Technologies

- HTML → Page structure
- CSS → Styling
- JavaScript → Logic
- Chart.js → Graphs
- Tailwind CSS → UI design

## Data Format

Each company has a JSON file inside `data/` folder.

Example:

```json
{
  "revenue": [100, 120, 130],
  "eps": [1.2, 1.3, 1.5],
  "netIncome": [20, 25, 30],
  "sentiment": [7.5, 8.0, 8.2]
}
