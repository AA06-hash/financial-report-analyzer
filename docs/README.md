# Financial Report Analyzer Documentation

Welcome to the full documentation for the Financial Report Analyzer project.

## Project Overview
The Financial Report Analyzer is a web-based dashboard that helps users analyze and visualize company financial data. It displays revenue, EPS, net income, risks, and sentiment using interactive charts.

## Project Scope
- Analyze financial data of multiple companies.
- Display charts for better understanding.
- Provide dark mode support.
- Show risk and sentiment analysis.
- Uses static JSON data (future API integration planned).

## Key Features
- Interactive charts using Chart.js
- Dark mode toggle
- Multi-company comparison
- Risk and sentiment visualization
- Easy-to-use dashboard

## Architecture
Technologies used in this project:

- Frontend: HTML, CSS, JavaScript
- Styling: Tailwind CSS
- Charts: Chart.js
- Data Source: JSON files
